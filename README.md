# Linux of African Farming 🌱

Open-source soil monitoring for African smallholder farmers.

**Cost:** $38 | **Tech:** Arduino + Soil Sensor + GSM Module | **Output:** SMS Alert

## Current Status
- ✅ Tinkercad simulation working
- ⏳ Hardware build pending first sponsor

## The Problem
A $500 commercial soil sensor costs more than a farmer's annual seed budget.

## The Solution
A $38 open-source alternative that sends SMS alerts when crops need water.
